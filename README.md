# .dotfiles

## Requirements

- Alacritty
- fzf
- lazygit
- neovim (nightly)
  - fd
  - npm
  - python3-venv
  - win32yank (for WSL)
  - xsel (for non-WSL)
- ripgrep
- stow (v2.4.0+)
- tmux
- zoxide

## Installation

```bash
sudo apt-get install stow -y
git clone <https://github.com/declancm/.dotfiles> ~/.dotfiles
cd ~/.dotfiles && ./install.sh
```
